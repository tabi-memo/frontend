# Frontend repository for Tabi-memo

## Command

Run development

```bash
pnpm dev
```

Install dependencies

```bash
pnpm install
```

Run lint

```bash
pnpm lint
```

Run format

```bash
pnpm format:fix
```

Run storybook

```bash
pnpm storybook
```
