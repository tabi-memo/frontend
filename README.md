# Frontend repository for Tabi-memo

## Command

Run development

```bash
pnpm dev
```

Run Supabase local

- move to backend repository, and then run `supabase start`

```bash
supabase start
supabase stop
```

Install dependencies

```bash
pnpm install
```

Run lint

```bash
pnpm lint
```

Run format

```bash
pnpm format:fix
```

Run storybook

```bash
pnpm storybook
```

Run graphql-codegen

```bash
pnpm codegen
```
